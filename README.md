Overview
This project is a full-stack photo management application built with Spring Boot and React.js, designed to provide secure, scalable, and feature-rich user interactions.

Key Features
1. Photo Posting with Privacy Controls
Public Posts: Images are visible to all users.
Private Posts: Images are accessible only to the uploader.
2. Engagement Features
Implemented comment and like services to enhance user interaction and engagement.
3. Real-Time Notifications
Integrated Apache Kafka for efficient and reliable real-time messaging.
4. Machine Learning-Powered Photo Organization
Currently working on a machine learning-based feature for automatic photo classification and organization:

If a matching folder exists, the photo is categorized and moved automatically.
If no matching folder exists, a new folder is dynamically created based on the classification results.
This feature leverages image classification techniques, improving both accuracy and speed, and showcases the integration of AI into real-world applications.
Technology Stack
Backend: Spring Boot with JWT authentication for secure API development.
Frontend: React.js styled with Bootstrap for a responsive and user-friendly interface.
Messaging: Apache Kafka for real-time notifications.
Machine Learning: Image classification techniques for photo organization.
Highlights
Combines full-stack development and microservices architecture with innovative AI-powered solutions.
Demonstrates expertise in building secure, scalable, and user-focused applications.
